🚀 OverviewCrimson Desert Save Guardian runs in the background and automatically creates versioned backups of your save folder at configurable intervals.
It includes a modern graphical interface, backup validation, restore options, and intelligent features like automatic save detection and language switching.


⭐ Installation
Download the .exe
Run the application
Click Auto Detect or select your save folder manually
Click Start Automatic Backup




🔥 Main Features💾 Automatic Backup System
Automatically backs up your save every X minutes
Fully configurable interval
Keeps a fixed number of backups (rotating system)
Old backups are automatically replaced


🧠 Smart Save Detection
Automatically detects Crimson Desert save folder
Works with default Windows paths
Manual override if needed


🔄 One-Click Restore
Restore any backup instantly
Restore latest healthy backup automatically
Safety backup created before restore


🧪 Backup Integrity Check
Verifies backups using checksum
Identifies corrupted or incomplete backups
Visual indicator (Healthy / Not Healthy)



🔍 Advanced Backup Management
Search and filter backups
Filter only healthy backups
Sort by name, date, size, or integrity
Double-click quick actions (restore / validate)
